Esterm permet de se connecter de manière automatique au Telit via un microcontrôleur, par l'USB ou le port série.

Voici un guide permettant de se servir de ce programme:

1) Tout d'abord, changez le paramètre 'path' dans parameters.ini, il vous permet de charger les .pyc et les .ini de ce dossier.

2) Le menu principal vous permez :
	- d'envoyer des commandes AT
	- de naviguer dans le menu.

3) Le sous-menu "1-AT commands" sample vous propose une liste de commandes pré-définies.

4) Ctrl+C permet de:
	- revenir au menu principal
	- redémarrer Esterm si on était dans le menu principal
